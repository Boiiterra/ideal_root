# Ideal root with Kotlin

Welcome to some simple console application I made.

## Requirements

+ Kotlin

## How to use

### Linux

```shell
git clone https://github.com/TerraBoii/ideal_root.git &&
cd ideal_root &&
./run
```

### Windows

```shell
git clone https://github.com/TerraBoii/ideal_root.git
```

And find way to run ./ideal_root/src/main.kt
